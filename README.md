# scriptscrapper
IMSDB scrapper

Movie script scrapping script from imsdb


## Install & Run 
- Clone 
    ```
    git clone https://github.com/pj8912/scriptscrapper
    ```

- Create Virtual Environment

- Activate Environment

- Install Requirements

    ```
    pip install -r requirements.txt
    ```
- Run 
    ```
    python main.py
    ```
